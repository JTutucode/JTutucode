  <h2>Core Project</h2>

  <h3>LearnMind AI</h3>
  <p>
    LearnMind AI is an <b>adaptive AI-powered learning assistant</b> designed to personalize education through
    vector embeddings, semantic memory retrieval, and intelligent study-session recommendations.
  </p>

  <p>
    The system analyzes user learning history, retrieves relevant educational context, and dynamically adjusts
    study content to improve learning outcomes. It was built to support thousands of educational interactions
    while maintaining reliable and consistent backend workflows.
  </p>

  <p>
    LearnMind AI leveraged <b>vector embeddings</b> and <b>semantic memory retrieval</b> across 5,000+ educational
    interactions, increasing content recommendation relevance by 35% and improving user retention during testing
    by 28%.
  </p>

  <p>
    The platform also implemented <b>concurrency-safe reward and inventory management workflows</b> supporting
    1,000+ simultaneous actions while maintaining 99.9% transaction consistency.
  </p>

  <p>
    <b>Role:</b> Software Engineer & AI Engineer — AI system design, backend development, semantic retrieval,
    recommendation logic, and scalable workflow engineering.
  </p>
